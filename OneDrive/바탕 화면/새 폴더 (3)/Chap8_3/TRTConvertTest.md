pytorch-ssd 폴더로 이동
```
cd pytorch-ssd
```
```
mkdir onnx
```

pth -> onnx 변환 (3~5분 정도 시간 소요)
```
cp models/ssd_example/* ./onnx
```
```
python3 onnx_export.py --model-dir=./onnx --input=ssd_cctv_sample.pth
```

완료되면 다음과 같이 변환된 onnx 파일 확인 가능
```
ls ./onnx/ssd-mobilenet.onnx
```
